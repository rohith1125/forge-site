// features

Everything you need, nothing you don't

Built for developers who live in the terminal. Forge is fast, keyboard-driven, and gets out of your way.

РџА

Streaming responses

See output token-by-token as the model generates. Real-time token speed displayed while streaming. Abort anytime with Ctrl+C.

­ЪЏа

Full file system tools

Read, write, edit files. Run shell commands. Search with glob and regex grep. Fetch URLs. Search the web. All built in, permission-gated.

­Ъћљ

Permission system

Every tool requires explicit permission. Choose allow, deny, or always-allow per session. File writes and shell commands need your approval.

­ЪЊІ

Task tracking

Ask forge to plan multi-step work. It writes a task list visible live in the UI Рђћ updated in real time as each step completes.

­ЪДа

Plan mode

Toggle /plan to make forge reason and outline steps without executing any tools. Perfect for complex tasks before committing.

­ЪњЙ

Session persistence

Every conversation is saved. Resume any previous session with /resume. Session title auto-set from your first message.

­ЪЌю

Context compaction

Long conversations eating your context window? /compact summarizes the history into a concise snapshot without losing key context.

­Ъје

8 themes

Forge, Dracula, Tokyo Night, Amber, Matrix, Ocean, Nord, Rose Pine. Switch live with /theme <name>. Persisted across sessions.

­ЪЊі

Token & cost tracking

Live token count, prompt vs completion split bar, and cost estimate shown in the status bar. See exactly what each conversation costs.

­Ъїљ

Web tools

Forge can fetch any URL and search DuckDuckGo. Great for pulling in docs, checking APIs, or researching while coding.

Рїе№ИЈ

Input history

Navigate previous commands with РєЉ / РєЊ. Full keyboard-driven workflow Рђћ no mouse needed. Ctrl+L clears screen instantly.

­Ъћї

Plugin & MCP support

Extend forge with custom tools via plugins or the Model Context Protocol. Connect any MCP server to add new capabilities.
