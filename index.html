// features

Everything you need, nothing you don't

Built for developers who live in the terminal. Forge is fast, keyboard-driven, and gets out of your way.

⚡

Streaming responses

See output token-by-token as the model generates. Real-time token speed displayed while streaming. Abort anytime with Ctrl+C.

🛠

Full file system tools

Read, write, edit files. Run shell commands. Search with glob and regex grep. Fetch URLs. Search the web. All built in, permission-gated.

🔐

Permission system

Every tool requires explicit permission. Choose allow, deny, or always-allow per session. File writes and shell commands need your approval.

📋

Task tracking

Ask forge to plan multi-step work. It writes a task list visible live in the UI — updated in real time as each step completes.

🧠

Plan mode

Toggle /plan to make forge reason and outline steps without executing any tools. Perfect for complex tasks before committing.

💾

Session persistence

Every conversation is saved. Resume any previous session with /resume. Session title auto-set from your first message.

🗜

Context compaction

Long conversations eating your context window? /compact summarizes the history into a concise snapshot without losing key context.

🎨

8 themes

Forge, Dracula, Tokyo Night, Amber, Matrix, Ocean, Nord, Rose Pine. Switch live with /theme <name>. Persisted across sessions.

📊

Token & cost tracking

Live token count, prompt vs completion split bar, and cost estimate shown in the status bar. See exactly what each conversation costs.

🌐

Web tools

Forge can fetch any URL and search DuckDuckGo. Great for pulling in docs, checking APIs, or researching while coding.

⌨️

Input history

Navigate previous commands with ↑ / ↓. Full keyboard-driven workflow — no mouse needed. Ctrl+L clears screen instantly.

🔌

Plugin & MCP support

Extend forge with custom tools via plugins or the Model Context Protocol. Connect any MCP server to add new capabilities.

// install

Installation guide

Single binary — no Node.js, no npm, no dependencies. Download, make executable, set your API key, run.

macOS Apple Silicon
macOS Intel
Linux
1
Download the binary

Open Terminal and run this command to download forge and place it in /usr/local/bin so it's available system-wide.

curl -L https://github.com/rohith1125/forge-releases/releases/latest/download/forge-mac-arm64 \
  -o /usr/local/bin/forge

Don't have curl? Install Xcode Command Line Tools first: xcode-select --install

2
Make it executable

Grant execute permission to the binary.

chmod +x /usr/local/bin/forge
3
Bypass macOS Gatekeeper macOS only

macOS blocks unsigned binaries by default. Run this once to remove the quarantine flag:

xattr -d com.apple.quarantine /usr/local/bin/forge

Alternative: If the above doesn't work, go to System Settings → Privacy & Security, scroll to the bottom, and click "Allow Anyway" next to the forge warning.

4
Get an API key

Forge needs an API key from Anthropic (Claude) or OpenAI (GPT-4). Choose one:

🟣 Anthropic (recommended)

Claude Sonnet 4 — best balance of speed and intelligence.

  1. Go to console.anthropic.com
  2. Sign up / log in
  3. Click API Keys → Create Key
  4. Copy the key starting with sk-ant-...
🟢 OpenAI

GPT-4o and variants.

  1. Go to platform.openai.com
  2. Sign up / log in
  3. Click API Keys → Create new secret key
  4. Copy the key starting with sk-...
5
Set your API key permanently

Add your key to your shell profile so it's set automatically every session. Replace sk-ant-YOUR-KEY with your actual key.

# For Anthropic (Claude)
echo 'export ANTHROPIC_API_KEY=sk-ant-YOUR-KEY' >> ~/.zshrc
source ~/.zshrc

# OR for OpenAI
echo 'export OPENAI_API_KEY=sk-YOUR-KEY' >> ~/.zshrc
source ~/.zshrc

Using bash instead of zsh? Replace ~/.zshrc with ~/.bash_profile

6
Launch forge

Open a new terminal window (so the env var is loaded) and run:

forge

You should see the forge UI with the ASCII logo. Type a message and press Enter to start. Type /help for all commands.

// troubleshooting
❯ forge: command not found

/usr/local/bin may not be in your PATH. Fix: export PATH="/usr/local/bin:$PATH" and add that line to your ~/.zshrc or ~/.bashrc. Or move the binary to ~/bin/forge and add ~/bin to PATH.

❯ macOS: "forge cannot be opened because the developer cannot be verified"

Run: xattr -d com.apple.quarantine /usr/local/bin/forge
Or: System Settings → Privacy & Security → scroll down → Allow Anyway.

❯ Error: ANTHROPIC_API_KEY is not set

Make sure you sourced your profile after adding the key: source ~/.zshrc. Or set it inline: ANTHROPIC_API_KEY=sk-ant-... forge

❯ forge: requires an interactive terminal (TTY)

Forge requires a real terminal. Don't run it via shell scripts, piped input, or non-TTY environments. Open iTerm2, Terminal.app, or any real terminal emulator.

/resume [id]Resume a previous saved session
/model [name]Switch AI model mid-conversation
/exitQuit forge
